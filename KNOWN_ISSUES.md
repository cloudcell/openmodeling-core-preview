# Known issues

This is an alpha preview. The following issues are known and expected.

## General

- Incomplete documentation and UI copy.
- Some error messages are not user-friendly.
- Performance has not been optimized for large models.
- Keyboard shortcuts and accessibility features are limited.

## Modeling

- Some edge cases in rule evaluation may produce unexpected results.
- Undo/redo may not cover all actions.
- Import and export formats are limited.

## Platform

- Not all operating systems are supported yet.
- Installation packages are not signed.

## Reporting new issues

Please open a [GitHub issue](../../issues) or report on [Discord](https://discord.gg/GfU5ypAbaD).
