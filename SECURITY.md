# Security

## Reporting security issues

Please do not report security issues publicly in GitHub issues or Discord.

Email security-sensitive reports to:

```text
alex@cloudcell.nz
```

Include a clear description of the issue, the affected build version, and steps to reproduce if possible.

## Response

We will acknowledge receipt and keep you informed of our progress.

## Scope

This policy applies to the preview builds of OM Core distributed in this repository.
