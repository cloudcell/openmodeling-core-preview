# Contributing

## Code contributions

Source code is not public yet, so code contributions are not open at this time.

## Other contributions

We welcome:

- Bug reports.
- Installation and usability feedback.
- Sample models and modeling examples.
- Product feedback and feature suggestions.

## How to contribute feedback

- Open a [GitHub issue](../../issues).
- Join the discussion on [Discord](https://discord.gg/GfU5ypAbaD).

## Code of conduct

Be respectful and constructive. We are building this in the open and value honest feedback.
