# Install and run

> These instructions are for the preview builds of OM Core. Only binary builds are provided here; source code is not public yet.

## Requirements

Check the release notes for the latest supported platforms.

## Download

1. Go to [GitHub Releases](../../releases).
2. Download the archive for your operating system.
3. Extract the archive to a folder of your choice.

## Run

Double-click the `OpenModeling` application, or run it from the extracted folder.

## First steps

Open a sample model from the [`samples/`](samples/) directory to see how dimensions, groups, cubes, and rules work.
